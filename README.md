# Daxmod

daxmod is a Python module for text classification

## Installation

### Dependencies

daxmod requires:

- Python (>=3.9)

- joblib (>=1.1.0)

- numpy (>=1.21.4)

- pandas (>=1.3.4)

- scikit-learn (>=1.0.1)

- tensorflow (>=2.7.0)

- tensorflow_hub (>=0.12.0)

- tensorflow_text (>=2.7.3)

### User installation

```bash
pip install -U daxmod
```

### Source code

```bash
git clone https://github.com/Authentic10/daxmod.git
```
